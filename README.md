Sports Center Pdr


Descrição do Projeto:
O Sports Center Pdr é um site de uma loja virtual de produtos esportivos, desenvolvido para a disciplina de Programação Web.
Essa loja virtual tem itens à venda como tênis, camisas e bolas de esportes, tudo envolvido a meios esportivos, como tênis para praticar esportes, camisas apropriadas a treinos e bolas de futebol e basquete.
O projeto foi desenvolvido utilizando HTML5,CSS, JavaScript e Bootstrap. Através do Bootstrap minimizou o uso de CSS e facilitou o desenvolvimento do layout. 


Página Online

O site está disponível online através do GitHub Pages:
https://renannunesds.github.io/SportsCenterPdr/


Funcionalidades do Site:
- Navegação pelo Navbar fixed

- Seção Hero com mensagens e botão de promoção

- Seção Sobre explicando o que é o site e o objetivo

- Produtos de esportes em exibição

- Cada produto tem o botão de adicionar no carrinho

- Carrinho de compras com todos os produtos adicionados

- Cálculo da soma de todos os produtos que está no carrinho

- Botão Promoção do Dia que dá 10% de desconto nos produtos

- Seção Contato com informações de comunicação e redes sociais

- Layout responsivo utilizando Bootstrap


Processo de Desenvolvimento:
O processo começou com a criação da estrutura do HTML com foco em semântica. A parte visual, como o layout, foi desenvolvido a partir do Bootstrap em que facilitou o processo, e foi utilizado um pouco de CSS. Após isso, os produtos do site foram organizados em 3 categorias, com tênis, camisas e bolas de esportes. Cada produto tinha um botão de enviar ao carrinho, em que a parte do carrinho de compras foi feita através do JavaScript. O JS está presente também na implementação da promoção no site. Foi feito alguns ajustes na barra de navegação ao longo do desenvolvimento, alteração nas imagens, ajustes no tamanho das imagens e na responsividade para garantir uma boa experiência independente do dispositivo do usuário. Após todo o processo no VS Code, o site foi publicado no GitHub Pages.



Facilidades Encontradas:
- Utilização do Bootstrap, que facilitou a criação do layout

- Alguns componentes prontos como navbar, cards e botões

- Separar entre HTML e JavaScript facilitou a organização do código



Dificuldades Encontradas:
- Controle do carrinho de compras com itens diferentes usando JavaScript

- Atualização o valor total do carrinho com o desconto da promoção

- Padronizar todas as imagens com o mesmo tamanho

- Manipulação do DOM para sincronizar o carrinho com a exibição visual
